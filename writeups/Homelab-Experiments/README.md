# 🧪 Homelab Offensive & Defensive Testing

## 📋 Overview
This directory logs empirical security research conducted directly against the local homelab infrastructure. It acts as a laboratory for testing defensive controls, evaluating firewall mechanics, and identifying misconfigurations.

## 🔬 Focus Areas
* **Firewall Evasion & Policy Testing:** Assessing OPNsense rule efficiency and validating inter-vlan blocks.
* **IDS/IPS Tuning:** Triggering and filtering alerts (Suricata/Snort) to optimize signal-to-noise ratio against automated scanning tools.
* **Traffic Analysis:** Parsing raw communication protocols to study normal baseline behavior versus anomalous network traffic.
